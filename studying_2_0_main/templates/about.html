What is this website about?

this website deals with the problem of structuring your data properly. The purpose for this might be differ to the users problem. For example saving, managing and structuring your materials in a digital way for university, school or even something different. When saving it on this website you are able to access it from everywhere and work with it easily. As the website is still under construction many features are currently not available but will be added soon. To learn more about it and how to properly use it you can go the help section.
